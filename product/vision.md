# Product Vision

**Status:** Stage 0 — Concept Validated  
**Stage:** 0

---

## What Is It

AgentOS Continuum is a local-first AI companion platform for personal workflow memory, learning continuity, and proactive guidance.

Stareha is the companion inside AgentOS Continuum.

```
AgentOS Continuum = the system (body)
Stareha = the companion inside (mind/personality)
```

---

## Vision Statement

> Stareha is a private AI companion that learns how you work, remembers what matters, and prepares personalized guidance so you can continue where you left off.

## Mission

> Help people learn, build, and work with continuity.

## Main Promise

```
Never restart from zero.
```

Every time the user opens their laptop, Stareha should know:
- What the user was doing
- What they were trying to learn
- What they struggled with
- What they completed
- What the next useful step is
- What resources or exercises can help
- What should be suggested now

---

## What It Is NOT

- Not another chatbot
- Not a coding agent
- Not a CLI assistant
- Not a productivity tracker

It is a companion that lives with the user's work context.

---

## End-Goal Experience

```
The user opens their laptop.

Stareha already knows:
- what they were working on
- what they were learning
- where they got stuck
- what they should do next
- what exercises or resources will help today

Stareha says:
"I prepared your next step."
```

---

## Target Users (First)

- Developers
- Indie hackers
- Linux power users
- Students learning programming
- Research-heavy builders
- People using terminal/browser/AI tools daily

---

## One-Line Descriptions

**Simple:** Stareha is your AI companion that remembers your work and prepares your next step.

**Developer-focused:** Stareha is a local-first workflow memory companion for developers.

**Student-focused:** Stareha is a personal AI mentor that learns how you learn and prepares custom practice.

**Full product:** AgentOS Continuum gives your computer memory. Stareha turns that memory into guidance.
