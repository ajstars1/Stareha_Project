# Naming & Relationship

**Status:** Defined  
**Stage:** 0

---

## AgentOS Continuum

The broader platform, system, and runtime.

Includes:
- Local memory
- Workflow learning
- Learning ledger
- Daemon/runtime
- CLI
- Desktop interface
- Browser/Android extensions (later)
- Local and cloud intelligence
- Permission system
- Sync layer (later)

---

## Stareha

The personal AI companion inside AgentOS Continuum.

Stareha is the user-facing identity.

It:
- Talks to the user
- Remembers their progress
- Prepares learning material
- Suggests next steps
- Acts as a mentor or work companion

---

## The Relationship

```
AgentOS Continuum = the system
Stareha = the companion inside the system
```

```
AgentOS Continuum is the body.
Stareha is the mind/personality.
```
