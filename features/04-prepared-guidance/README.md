# Feature: Prepared Guidance

**Status:** Built  
**Stage:** 4  
**Why it matters:** This is what makes Stareha feel like a mentor, not a logger. Proactive preparation is the product's most powerful experience.

---

## What It Is

Prepared Guidance is Stareha's ability to analyze the user's recent work, detect gaps and weak areas, and prepare personalized practice or work plans before the next session starts.

The user doesn't ask. Stareha prepares.

---

## Why It Matters

The mentor experience:

Without Stareha:
```
User opens laptop.
Thinks: "What was I doing? Where was I? What should I study?"
Spends 20 minutes reorienting.
```

With Stareha:
```
User opens laptop.
Stareha: "Good morning. I prepared today's plan based on yesterday."
User starts immediately with exactly what they need.
```

## How It Feels (Example)

```
stareha continue

Tomorrow's plan:

🎯 Goal: CSS Flexbox (you've been struggling with this)

1. Recap (10 min)
   flexbox-recap.md (prepared)

2. Quiz (5 min)
   5 questions: justify-content, align-items, flex-direction

3. Exercise (30 min)
   Build a responsive pricing card

4. Stretch: DOM selectors (3 short tasks)
   (you've hit errors here 4 times this week)
```

Prepared guidance is generated automatically by `stareha done` and can be refreshed manually with `stareha prep`. The next beginner session started with `stareha learn "<goal>"` shows any pending briefing first.
